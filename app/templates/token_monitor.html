Token 优化策略（已实施）

1. Prompt 压缩：将 AI 提取 prompt 从冗长描述压缩为紧凑指令格式，减少约 40% input tokens。

2. 智能跳过：当规则引擎置信度 >= 0.95 时，跳过 AI 调用，直接使用规则结果。典型命中率约 30-40%。

3. 结果缓存：对相同公告内容（MD5 去重）缓存 AI 提取结果，避免重复处理同一公告。

4. 输出上限：max_tokens 从 1200 降至 800，单次事件提取足够且减少 output tokens。

5. 共识机制：规则与 AI 结果一致时置信度提升 +0.02，减少人工审核量，间接节省重试 token。
